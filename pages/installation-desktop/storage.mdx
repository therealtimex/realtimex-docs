# General information

## Where is my data located?

All data pertaining to RealTimeX Desktop will be in the following locations. Please replace `<usr>` with your
device username.

On Mac:
`/Users/<usr>/.realtimex.ai/storage`

On Linux:
`~/.config/.realtimex.ai/storage/`

On Windows:
`C:\Users\<usr>\AppData\Roaming\.realtimex.ai\storage`

### What is each folder?

- `lancedb`: This it where your local vector database and its tables are stored.
- `documents`: This is the parsed document content of any uploaded files.
- `vector-cache`: This folder is the _cached_ and embedded representation of a previous uploaded and embedded file. Its filename is hashed.
- `models`: Any locally stored LLMs or Embedder models used by the system are stored here. Typically are GGUF files.
- `RealTimeX.db`: This is the RealTimeX SQLite database.
- `plugins`: This is the folder where your custom agent skills are stored.
