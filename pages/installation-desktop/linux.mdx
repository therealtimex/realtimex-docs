# Linux Installation

**1. Install using the [installer script](#install-using-the-installer-script)**

## Install using the installer script

<Callout type="warning" emoji="⚠️">
  <strong>NOTICE</strong><br /><br />
  ➤ RealTimeX packages as an AppImage but you <strong>will not</strong> be able to boot if you run just the AppImage. Please only use the script below.<br /><br />
  ➤ Please open a <a href="https://github.com/Mintplex-Labs/anything-llm/issues" target="_blank">Github Issue</a> if you have installation or bootup troubles.
</Callout>

First, open a terminal on your Linux machine and run this command.

```shell copy
curl -fsSL https://cdn.realtimex.ai/latest/installer.sh | sh
```

You can view the raw script contents [here](https://github.com/Mintplex-Labs/anything-llm/blob/desktop/linux_installer.sh).

This will download the latest version of RealTimeX's AppImage, unpack it, and then supply a symlink to seamlessly run RealTimeX.
This script will unpack the app in $HOME/RealTimeXDesktop.

You can start the app at any time by running ./RealTimeXDesktop/start. This will boot the app with full logging.
