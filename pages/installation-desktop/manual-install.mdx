# Install Ollama for RealTimeX Desktop

To use local LLM capabilities in RealTimeX Desktop, you need to have [Ollama](https://ollama.com/) installed on your system. This guide will walk you through the straightforward process of installing Ollama directly.

## Step 1: Download Ollama

- Visit the [Ollama download page](https://ollama.com/) and download the latest installer.

## Step 2: Install Ollama

- Run the downloaded installer and follow the on-screen instructions to complete the installation.

## Step 3: Configure RealTimeX Desktop

- Open RealTimeX Desktop.
- In the application settings, select Ollama as your LLM provider.

## Troubleshooting

- If you encounter any issues during installation, refer to the [Ollama documentation](https://github.com/ollama/ollama/tree/main/docs) or reach out to RealTimeX support.
