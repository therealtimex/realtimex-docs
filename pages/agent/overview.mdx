# AI Agents

AI Agents in RealTimeX are LLM-powered assistants with access to tools, automations, and integrations.  
Agents operate within your workspace but can be invoked in any chat or workflow via `@agent`.

<Callout type="info" emoji="ðŸ¤–">
  **Good to know:**  
  All agents share the same toolset across workspaces, but can behave differently depending on their workspace context.  
  More customization and advanced agent features are coming soon!
</Callout>
